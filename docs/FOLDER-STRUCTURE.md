## Route Group Summary

| Group | Path prefix | Layout |
|-------|-------------|--------|
| `(auth)` | `/sign-in`, `/sign-up` | Minimal centered |
| `(shop)` | `/`, `/products`, `/cart`, вЂ¦ | Customer header/footer |
| `(admin)` | `/admin/*` | Sidebar + admin guard |

## File Naming Conventions

- **Pages**: `page.tsx`, `layout.tsx`, `loading.tsx`, `error.tsx`
- **Components**: `kebab-case.tsx`, named exports PascalCase
- **Actions**: `verb-noun` functions in `actions/*.ts` with `"use server"`
- **Validations**: `*.ts` exporting Zod schemas + inferred types
